# Code and exercises reading Learn you a Haskell

## Link to book
[Learn You a Haskell for Great Good!](http://learnyouahaskell.com)

## Run
Run tests with:

``` sh
make test
```

Execute the app with

``` sh
cabal run learn-you-a-haskell
```

or 

``` sh
make run
```
